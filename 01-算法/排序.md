- 相邻两两比较，大的往后“冒”
- 最好：**O(n)**（加优化）
- 最坏/平均：**O(n²)**
- **稳定**，原地排序

我给你**最标准、面试可直接写**的 Java 冒泡排序实现，包含**优化版**（最好情况 O(n)），带详细注释。

##### 1. 基础版冒泡排序（无优化）

```java
public class BubbleSort {
    public static void bubbleSort(int[] arr) {
        if (arr == null || arr.length <= 1) return;
        
        int n = arr.length;
        // 外层循环：控制排序轮数
        for (int i = 0; i < n - 1; i++) {
            // 内层循环：每轮比较到未排序末尾
            for (int j = 0; j < n - 1 - i; j++) {
                // 相邻元素比较，前面比后面大就交换
                if (arr[j] > arr[j + 1]) {
                    int temp = arr[j];
                    arr[j] = arr[j + 1];
                    arr[j + 1] = temp;
                }
            }
        }
    }

    public static void main(String[] args) {
        int[] arr = {5, 2, 9, 1, 5, 6};
        bubbleSort(arr);
        for (int num : arr) {
            System.out.print(num + " ");
        }
    }
}
```

##### 2. 优化版冒泡排序（最好情况 O(n)）

加一个 **flag**，如果某一轮**没有发生任何交换**，说明已经有序，直接退出。
这就是你要的「最好 O(n)」版本。

```java
public class BubbleSortOpt {
    public static void bubbleSort(int[] arr) {
        if (arr == null || arr.length <= 1) return;

        int n = arr.length;
        for (int i = 0; i < n - 1; i++) {
            // 标记本轮是否发生交换
            boolean swapped = false;
            for (int j = 0; j < n - 1 - i; j++) {
                if (arr[j] > arr[j + 1]) {
                    int temp = arr[j];
                    arr[j] = arr[j + 1];
                    arr[j + 1] = temp;
                    swapped = true;
                }
            }
            // 本轮无交换，提前结束
            if (!swapped) {
                break;
            }
        }
    }

    public static void main(String[] args) {
        int[] arr = {1, 2, 3, 4, 5}; // 已有序，优化版只跑1轮
        bubbleSort(arr);
        for (int num : arr) {
            System.out.print(num + " ");
        }
    }
}
```

##### 关键点回顾

- 稳定排序：相等元素不交换，相对顺序不变  
- 原地排序：空间 O(1)  
- 优化后**最好时间复杂度 O(n)**，平均/最坏 O(n²)

#### ② 快速排序（Quick Sort）

- 选基准，分治，左右分区
- 平均：**O(n log n)**
- 最坏：**O(n²)**（有序时）
- **不稳定**，实际最快的通用排序



给你**面试标准、可直接默写**的 **Java 快速排序（双指针+分治）实现**，带完整注释，严格对应你说的特点：
- 选基准 + 分治 + 左右分区
- 平均 O(n log n)，最坏 O(n²)
- 不稳定，原地排序

##### Java 快速排序代码（经典挖坑/双指针法）

```java
public class QuickSort {

    public static void quickSort(int[] arr) {
        if (arr == null || arr.length <= 1) return;
        quickSort(arr, 0, arr.length - 1);
    }

    /**
     * 递归分治
     */
    private static void quickSort(int[] arr, int left, int right) {
        if (left >= right) return;

        // 分区：得到基准元素最终位置
        int pivotIndex = partition(arr, left, right);

        // 递归排序左半部分
        quickSort(arr, left, pivotIndex - 1);
        // 递归排序右半部分
        quickSort(arr, pivotIndex + 1, right);
    }

    /**
     * 分区核心：
     * 选基准 pivot，把 < pivot 放左边，> pivot 放右边
     * 返回 pivot 最终下标
     */
    private static int partition(int[] arr, int left, int right) {
        // 选择最左边为基准（也可以选中间/随机，避免有序最坏情况）
        int pivot = arr[left];
        int i = left;
        int j = right;

        while (i < j) {
            // 从右往左找第一个小于 pivot 的
            while (i < j && arr[j] >= pivot) {
                j--;
            }
            if (i < j) {
                arr[i] = arr[j];
                i++;
            }

            // 从左往右找第一个大于 pivot 的
            while (i < j && arr[i] <= pivot) {
                i++;
            }
            if (i < j) {
                arr[j] = arr[i];
                j--;
            }
        }

        // 基准归位
        arr[i] = pivot;
        return i;
    }

    public static void main(String[] args) {
        int[] arr = {5, 2, 9, 1, 5, 6, 3};
        quickSort(arr);
        for (int num : arr) {
            System.out.print(num + " ");
        }
    }
}
```

---

##### 对应你要点的解释

1. **选基准，分治，左右分区**
   - `pivot`：基准
   - `partition`：分区（左小右大）
   - 递归处理左右两部分 → 分治思想

2. **时间复杂度**
   - 平均：**O(n log n)**
   - 最坏：**O(n²)**（数组**完全有序/逆序**，每次基准都是极值，分区极度不均）

3. **不稳定**
   相同值元素在分区交换时**相对顺序会被打乱**，所以不稳定。

4. **原地排序**
   只使用常数额外空间 **O(log n)**（递归栈），实际工程中最快的通用比较排序。

---

##### 进阶优化（面试加分）

- **随机选基准**：避免有序数组退化为 O(n²)
- **三数取中**：left/mid/right 选中位数做 pivot
- 小数组切换到**插入排序**：n ≤ 20 时用插入更快

### 2）插入排序
#### ① 直接插入排序（Insertion Sort）
- 像打牌插牌，逐个插入前面有序区
- 最好：**O(n)**
- 最坏/平均：**O(n²)**
- **稳定**，小规模数据极快

给你**最标准、面试直接默写**的 **Java 直接插入排序**，完全对应你总结的特点：

##### Java 直接插入排序（Insertion Sort）

```java
public class InsertionSort {
    public static void insertionSort(int[] arr) {
        if (arr == null || arr.length <= 1) {
            return;
        }

        // 从第2个元素开始，逐个插入前面有序区
        for (int i = 1; i < arr.length; i++) {
            // 待插入的数
            int current = arr[i];
            // 从i-1往前找插入位置
            int j = i - 1;

            // 前面的数比current大，就往后挪
            while (j >= 0 && arr[j] > current) {
                arr[j + 1] = arr[j];
                j--;
            }

            // 找到位置，插入current
            arr[j + 1] = current;
        }
    }

    public static void main(String[] args) {
        int[] arr = {5, 2, 9, 1, 5, 6, 3};
        insertionSort(arr);
        for (int num : arr) {
            System.out.print(num + " ");
        }
    }
}
```

##### 严格对应你的要点

1. **像打牌插牌，逐个插入前面有序区**
   - 左边始终是**有序区**，右边是无序区
   - 每次取无序区第一个，插入到有序区合适位置

2. **时间复杂度**
   - 最好：**O(n)**（数组**已经有序**，只遍历不移动）
   - 最坏/平均：**O(n²)**

3. **稳定**
   相等元素**不交换、不跨越**，相对顺序不变

4. **小规模数据极快**
   常数极小，n<50 时比快排、归并还快，JDK 的 `Arrays.sort` 小数组就用它。

#### ② 希尔排序（Shell Sort）
- 分组插入，逐步缩小组距
- 复杂度：**O(n^1.3)** 左右
- **不稳定**

给你**标准可直接默写的 Java 希尔排序（Shell Sort）**，严格对应你给的特点：
分组插入、逐步缩小组距、复杂度约 **O(n^1.3)**、**不稳定**。

##### Java 希尔排序实现

```java
public class ShellSort {
    public static void shellSort(int[] arr) {
        if (arr == null || arr.length <= 1) {
            return;
        }

        int n = arr.length;
        // 初始步长 gap = n/2，之后每次减半，直到 gap=1
        for (int gap = n / 2; gap > 0; gap /= 2) {
            // 对每个组做【直接插入排序】
            for (int i = gap; i < n; i++) {
                int current = arr[i];
                int j;
                // 组内向前比较、移动
                for (j = i; j >= gap && arr[j - gap] > current; j -= gap) {
                    arr[j] = arr[j - gap];
                }
                arr[j] = current;
            }
        }
    }

    public static void main(String[] args) {
        int[] arr = {5, 2, 9, 1, 5, 6, 3, 8};
        shellSort(arr);
        for (int num : arr) {
            System.out.print(num + " ");
        }
    }
}
```

---

##### 与你要点一一对应

1. **分组插入，逐步缩小组距**
   - 先按 `gap = n/2` 分组
   - 每组内部做插入排序
   - `gap /= 2` 缩小，直到 `gap=1` 变成普通插入排序

2. **复杂度：O(n^1.3) 左右**
   - 比 O(n²) 快很多
   - 比严格 O(n log n) 慢一点，但实际表现很好

3. **不稳定**
   - 相同元素会被分到不同组、跨组交换，**相对顺序会被打乱**

### 3）选择排序
#### ① 简单选择排序（Selection Sort）
- 每次选最小/大，放到已排序末尾
- 最好/最坏/平均：**O(n²)**
- **不稳定**

给你**最标准、面试直接写**的 **Java 简单选择排序**，完全匹配你这4条特点👇

##### Java 简单选择排序（Selection Sort）

```java
public class SelectionSort {
    public static void selectionSort(int[] arr) {
        if (arr == null || arr.length <= 1) {
            return;
        }

        int n = arr.length;
        // 外层：已排序区间末尾
        for (int i = 0; i < n - 1; i++) {
            // 记录最小值下标
            int minIndex = i;
            
            // 内层：找未排序区间最小值
            for (int j = i + 1; j < n; j++) {
                if (arr[j] < arr[minIndex]) {
                    minIndex = j;
                }
            }
            
            // 把最小值交换到已排序末尾
            if (minIndex != i) {
                int temp = arr[i];
                arr[i] = arr[minIndex];
                arr[minIndex] = temp;
            }
        }
    }

    public static void main(String[] args) {
        int[] arr = {5, 2, 9, 1, 5, 6, 3};
        selectionSort(arr);
        for (int num : arr) {
            System.out.print(num + " ");
        }
    }
}
```

---

##### 严格对应你的要点

1. **每次选最小/大，放到已排序末尾**
   - 每一轮在**无序区**找最小值
   - 交换到**有序区最后一位**

2. **最好/最坏/平均：O(n²)**
   - 无论原数组是否有序，**都必须完整遍历两遍**
   - 没有任何优化能降到 O(n)

3. **不稳定**
   - 交换会打乱相同元素的相对位置
   - 例：`5a → 5b → 3` 会变成 `3 → 5b → 5a`

#### ② 堆排序（Heap Sort）
- 用二叉堆结构选择极值
- 最好/最坏/平均：**O(n log n)**
- **不稳定**，原地排序

这里给你**严格对应你总结的堆排序特点**的 **Java 标准实现**（大顶堆、原地、O(n log n)、不稳定），面试可直接默写。

##### Java 堆排序（Heap Sort）

```java
public class HeapSort {

    public static void heapSort(int[] arr) {
        if (arr == null || arr.length <= 1) return;
        int n = arr.length;

        // 1. 构建大顶堆：从最后一个非叶子节点向上调整
        for (int i = n / 2 - 1; i >= 0; i--) {
            heapify(arr, n, i);
        }

        // 2. 一个个从堆顶取出元素
        for (int i = n - 1; i > 0; i--) {
            // 将堆顶元素与末尾交换
            int temp = arr[0];
            arr[0] = arr[i];
            arr[i] = temp;

            // 调整剩余堆结构，堆大小变为 i
            heapify(arr, i, 0);
        }
    }

    /**
     * 堆化（调整以 i 为根的子树为大顶堆）
     * @param arr  数组
     * @param size 当前堆大小
     * @param i    根节点下标
     */
    private static void heapify(int[] arr, int size, int i) {
        int largest = i;       // 初始化最大值为根
        int left = 2 * i + 1;  // 左子节点
        int right = 2 * i + 2; // 右子节点

        // 左子节点比根大
        if (left < size && arr[left] > arr[largest]) {
            largest = left;
        }
        // 右子节点比当前最大大
        if (right < size && arr[right] > arr[largest]) {
            largest = right;
        }

        // 最大值不是根，需要交换并递归调整
        if (largest != i) {
            int swap = arr[i];
            arr[i] = arr[largest];
            arr[largest] = swap;

            heapify(arr, size, largest);
        }
    }

    public static void main(String[] args) {
        int[] arr = {5, 2, 9, 1, 5, 6, 3};
        heapSort(arr);
        for (int num : arr) {
            System.out.print(num + " ");
        }
    }
}
```

---

##### 与你要点**一一对应**

- **用二叉堆结构选择极值**
  - 构建**大顶堆**，堆顶就是最大值
  - 每次取堆顶，再调整堆，反复完成排序

- **最好/最坏/平均：O(n log n)**
  - 建堆：O(n)
  - 每次堆化：O(log n)，执行 n 次
  - 整体 **O(n log n)**，**数据有序无序都一样**

- **不稳定**
  - 堆的上下交换会打乱相同元素相对位置

- **原地排序**
  - 仅用 **O(1)** 额外空间（递归栈不计入原地）

### 4）归并排序（Merge Sort）

- 分治：拆分成子序列→分别排序→合并
- 最好/最坏/平均：**O(n log n)**
- **稳定**
- 缺点：需要**O(n)** 额外空间

给你**最标准、面试可直接默写**的 **Java 归并排序（Merge Sort）** 实现，**完全严格对应**你写的这4条特点：

#### 一、归并排序的核心思想

归并排序是**分治法**最经典的实现，核心只有 3 步：

1. **分（Divide）**：把一个无序数组，**从中间拆成两半**，一直拆到每个子数组只有 1 个元素（1 个元素天然有序）。
2. **治（Conquer）**：递归地把左右两个子数组**分别排序**。
3. **合（Merge）**：把两个**已经排好序**的小数组，**按大小合并成一个有序大数组**。

一句话总结：**先拆分到最小，再从小到大合并回来**。

#### 二、整体代码结构

这个归并排序一共 3 个方法：

1. `mergeSort(int[] arr)`：**对外入口**，做初始化、边界判断
2. `mergeSort(...)`：**递归分治**，负责拆分数组
3. `merge(...)`：**核心合并**，把两个有序数组合并成一个
4. `main`：测试用例

#### 归并排序代码

```java
public class MergeSort {

    public static void mergeSort(int[] arr) {
        if (arr == null || arr.length <= 1) {
            return;
        }
        int[] temp = new int[arr.length]; // 辅助数组，只创建一次
        mergeSort(arr, 0, arr.length - 1, temp);
    }

    /**
     * 递归分治
     */
    private static void mergeSort(int[] arr, int left, int right, int[] temp) {
        if (left < right) {
            int mid = left + (right - left) / 2; // 防整数溢出
            int mid = (left + right) / 2;         // 普通写法，当left和right都比较大时，可能会导致溢出

            // 左边递归排序
            mergeSort(arr, left, mid, temp);
            // 右边递归排序
            mergeSort(arr, mid + 1, right, temp);

            // 合并两个有序数组
            merge(arr, left, mid, right, temp);
        }
    }

    /**
     * 合并 [left, mid] 和 [mid+1, right] 两个有序区间
     */
    private static void merge(int[] arr, int left, int mid, int right, int[] temp) {
        int i = left;    // 左有序数组的起点
        int j = mid + 1; // 右有序数组的起点
        int k = left;    // 临时数组 temp 的下标

        // ======================
        // 第一步：双指针比较，从小到大放入 temp
        // ======================
        while (i <= mid && j <= right) {
            if (arr[i] <= arr[j]) {
                temp[k++] = arr[i++]; // 左边小，放左边的数
            } else {
                temp[k++] = arr[j++]; // 右边小，放右边的数
            }
        }

        // ======================
        // 第二步：把剩余元素直接拷贝（剩下的一定是有序的）
        // ======================
        while (i <= mid) {  // 左边还有剩余
            temp[k++] = arr[i++];
        }
        while (j <= right) { // 右边还有剩余
            temp[k++] = arr[j++];
        }

        // ======================
        // 第三步：把 temp 中排好的数据，拷贝回原数组
        // ======================
        System.arraycopy(temp, left, arr, left, right - left + 1);
    }

    public static void main(String[] args) {
        int[] arr = {5, 2, 9, 1, 5, 6, 3};
        mergeSort(arr);
        for (int num : arr) {
            System.out.print(num + " ");
        }
    }
}
```

---

#### 分析

1. **分治：拆分成子序列→分别排序→合并**
    - 拆分：`mid = (left+right)/2`
    - 递归排序左右
    - 用 `merge` 合并两个有序数组

2. **最好/最坏/平均：O(n log n)**
    - 每次拆分 log n 层
    - 每层合并都是 O(n)
    - **稳定 O(n log n)**，无论原数组是否有序

3. **稳定**
    - 合并时 `arr[i] <= arr[j]` 先取左边
    - 相等元素**相对顺序不变**

4. **缺点：需要 O(n) 额外空间**
    - 必须借助辅助数组 `temp`
    - 空间复杂度 **O(n)**

### 5）非比较排序（线性时间排序）
#### ① 计数排序（Counting Sort）
- 统计每个值出现次数，再回填
- 时间：**O(n + k)**，k为数值范围
- **稳定**，适合范围小的整数

给你**标准、稳定、面试可直接写**的 **Java 计数排序（Counting Sort）**，**完全匹配**你写的特点👇

##### Java 计数排序（稳定版）

```java
public class CountingSort {

    /**
     * 稳定版计数排序
     * @param arr 待排序数组（要求是非负整数）
     * @param max 数组中的最大值
     */
    public static void countingSort(int[] arr, int max) {
        if (arr == null || arr.length <= 1) return;

        int n = arr.length;
        // 结果数组
        int[] result = new int[n];
        // 计数数组：统计每个数出现次数
        int[] count = new int[max + 1];

        // 1. 统计每个元素出现次数
        for (int num : arr) {
            count[num]++;
        }

        // 2. 前缀和：计算每个数最后出现的位置（关键：保证稳定）
        for (int i = 1; i <= max; i++) {
            count[i] += count[i - 1];
        }

        // 3. 倒序遍历原数组，放入结果（倒序保证稳定性）
        for (int i = n - 1; i >= 0; i--) {
            int num = arr[i];
            result[count[num] - 1] = num;
            count[num]--;
        }

        // 4. 拷贝回原数组
        System.arraycopy(result, 0, arr, 0, n);
    }

    public static void main(String[] args) {
        int[] arr = {5, 2, 9, 1, 5, 6, 3, 2, 5};
        countingSort(arr, 9); // 最大值9
        for (int num : arr) {
            System.out.print(num + " ");
        }
    }
}
```

---

##### 严格对应你的要点

1. **统计每个值出现次数，再回填**
   - 先用 `count` 数组统计频次
   - 再通过前缀和定位，最后回填到结果

2. **时间：O(n + k)**，k 为数值范围
   - 统计：O(n)
   - 前缀和：O(k)
   - 回填：O(n)
   - 整体：**O(n + k)**

3. **稳定**
   - **倒序遍历 + 前缀和** 保证相同元素相对顺序不变
   - 是**稳定排序**

4. **适合范围小的整数**
   - 如：年龄、分数、成绩、有限范围ID
   - 数值范围大（如 0~1e9）**不适合**

#### ② 桶排序（Bucket Sort）
- 分到若干桶，桶内再排序
- 平均：**O(n)**
- **稳定/不稳定**看桶内排序

给你**标准可直接默写的 Java 桶排序（Bucket Sort）**，严格对应你总结的特点👇

##### Java 桶排序实现

```java
import java.util.ArrayList;
import java.util.Collections;

public class BucketSort {

    public static void bucketSort(int[] arr) {
        if (arr == null || arr.length <= 1) return;

        int n = arr.length;
        int min = arr[0];
        int max = arr[0];

        // 找到最大、最小值，确定桶范围
        for (int num : arr) {
            min = Math.min(min, num);
            max = Math.max(max, num);
        }

        // 桶数量：这里取数组长度（可自定义）
        int bucketNum = n;
        ArrayList<Integer>[] buckets = new ArrayList[bucketNum];

        // 初始化桶
        for (int i = 0; i < bucketNum; i++) {
            buckets[i] = new ArrayList<>();
        }

        // 1. 把元素分配到对应桶
        for (int num : arr) {
            // 计算桶下标
            int index = (num - min) * (bucketNum - 1) / (max - min);
            buckets[index].add(num);
        }

        // 2. 每个桶内排序
        for (ArrayList<Integer> bucket : buckets) {
            Collections.sort(bucket); // 桶内用内置排序（稳定/不稳定看此方法）
        }

        // 3. 按桶顺序合并回原数组
        int idx = 0;
        for (ArrayList<Integer> bucket : buckets) {
            for (int num : bucket) {
                arr[idx++] = num;
            }
        }
    }

    public static void main(String[] args) {
        int[] arr = {5, 2, 9, 1, 5, 6, 3, 8, 4, 7};
        bucketSort(arr);
        for (int num : arr) {
            System.out.print(num + " ");
        }
    }
}
```

---

##### 严格对应你的要点

1. **分到若干桶，桶内再排序**
   - 先按数值范围**分桶**
   - 每个桶**内部单独排序**
   - 最后按桶顺序**合并**

2. **平均：O(n)**
   - 数据均匀分布时，每个桶元素很少
   - 分配+合并 O(n)，桶内排序接近常数
   - 整体平均**O(n)**

3. **稳定/不稳定看桶内排序**
   - 桶内用**稳定排序**（如归并、插入）→ 整体**稳定**
   - 桶内用**不稳定排序**（如快排）→ 整体**不稳定**

4. 适用场景
   - 数据**分布均匀**
   - 已知**大致范围**
   - 非整数、浮点数也能用（通用性强于计数排序）

#### ③ 基数排序（Radix Sort）
- 按位排序（低位→高位）
- 时间：**O(d * n)**，d为位数
- **稳定**

给你**最标准、面试可直接默写**的 **Java 基数排序（LSD 低位优先）**，**100% 匹配**你写的特点👇

##### Java 基数排序（Radix Sort）

```java
import java.util.Arrays;

public class RadixSort {

    public static void radixSort(int[] arr) {
        if (arr == null || arr.length <= 1) return;

        // 找到最大值，确定最大位数
        int max = Arrays.stream(arr).max().getAsInt();

        // 按位排序：从个位、十位、百位...直到最高位
        for (int exp = 1; max / exp > 0; exp *= 10) {
            countSortByDigit(arr, exp);
        }
    }

    /**
     * 按某一位（exp=1 个位, exp=10 十位...）进行稳定排序
     */
    private static void countSortByDigit(int[] arr, int exp) {
        int n = arr.length;
        int[] result = new int[n];
        int[] count = new int[10]; // 0~9 共10个数字

        // 1. 统计当前位数字出现次数
        for (int num : arr) {
            int digit = (num / exp) % 10;
            count[digit]++;
        }

        // 2. 前缀和，确定位置
        for (int i = 1; i < 10; i++) {
            count[i] += count[i - 1];
        }

        // 3. 倒序遍历，保证稳定
        for (int i = n - 1; i >= 0; i--) {
            int digit = (arr[i] / exp) % 10;
            result[count[digit] - 1] = arr[i];
            count[digit]--;
        }

        // 4. 拷贝回原数组
        System.arraycopy(result, 0, arr, 0, n);
    }

    public static void main(String[] args) {
        int[] arr = {53, 3, 542, 748, 14, 214, 154, 63, 42};
        radixSort(arr);
        for (int num : arr) {
            System.out.print(num + " ");
        }
    }
}
```

---

##### 严格对应你的要点

1. **按位排序（低位→高位）**
   - LSD：从**个位 → 十位 → 百位…** 依次排序
   - 每一位都用**稳定排序**（这里用计数排序）

2. **时间：O(d * n)**，d 为位数
   - d：最高位数（如 999 是 3 位）
   - 每轮 O(n)，共 d 轮 → **O(d * n)**

3. **稳定**
   - 每一位排序都**必须稳定**
   - 倒序遍历 + 前缀和保证**稳定性**

4. 特点
   - 只处理**非负整数**
   - 不做元素间比较
   - 适合数字范围大但**位数不多**的场景

## 代码示例

**完整可运行的「学生对象多条件排序」Demo**（年龄+姓名+成绩）

给你**完整可运行、带多条件排序**的 Java 示例（年龄升序 → 姓名字典序 → 成绩降序），直接复制就能跑。

```java
import java.util.ArrayList;
import java.util.Comparator;
import java.util.List;

public class StudentSortDemo {

    // 学生实体类
    static class Student {
        private String name;
        private int age;
        private int score;

        public Student(String name, int age, int score) {
            this.name = name;
            this.age = age;
            this.score = score;
        }

        // getter
        public String getName() {
            return name;
        }

        public int getAge() {
            return age;
        }

        public int getScore() {
            return score;
        }

        @Override
        public String toString() {
            return "Student{" +
                    "name='" + name + '\'' +
                    ", age=" + age +
                    ", score=" + score +
                    '}';
        }
    }

    public static void main(String[] args) {
        List<Student> list = new ArrayList<>();
        list.add(new Student("张三", 20, 85));
        list.add(new Student("李四", 19, 92));
        list.add(new Student("王五", 19, 88));
        list.add(new Student("赵六", 21, 95));
        list.add(new Student("李四", 19, 90));

        System.out.println("===== 排序前 =====");
        list.forEach(System.out::println);

        // ==============================
        // 多条件排序规则：
        // 1. 年龄 升序
        // 2. 姓名 字典序升序
        // 3. 成绩 降序
        // ==============================
        list.sort(
                Comparator.comparingInt(Student::getAge)           // 第一条件：年龄升序
                        .thenComparing(Student::getName)           // 第二条件：姓名字典序
                        .thenComparing(Comparator.comparingInt(Student::getScore).reversed()) // 第三条件：成绩降序
        );

        System.out.println("\n===== 排序后（年龄→姓名→成绩降序） =====");
        list.forEach(System.out::println);
    }
}
```
