# Java 面试 

严格按照**真实大厂面试官评分表**整理，**把你强调的「源码深度」「业务场景题」独立成核心模块**，不合并、不省略，全覆盖。

## 一、Java 基础语法（基本功）
1. 数据类型、包装类、自动拆装箱
2. String、StringBuffer、StringBuilder
3. 异常体系：编译期/运行时异常、try-catch-finally
4. 泛型、反射、注解、Lambda、Stream
5. IO、NIO、BIO、AIO 概念

## 二、集合框架 + **源码深度考察（核心必考）**
1. List：ArrayList、LinkedList 结构与源码
2. Map：HashMap JDK1.7/1.8 源码、哈希冲突、扩容、树化
3. ConcurrentHashMap 源码、分段锁/CAS+synchronized
4. HashSet、LinkedHashMap、TreeMap 原理
5. 线程安全集合对比
**👉 单独考察：源码理解、设计思想、坑点**

## 三、多线程与并发 + **源码深度考察**
1. Thread、Runnable、Callable、Future
2. synchronized 原理、锁升级、锁优化
3. volatile 语义、MESI、禁止重排序
4. CAS、ABA 问题、Unsafe
5. AQS 源码：ReentrantLock、CountDownLatch、Semaphore
6. 线程池 ThreadPoolExecutor 源码、参数、拒绝策略
7. 死锁、线程安全、可见性、有序性

## 四、JVM 虚拟机（中高级分水岭）
1. JVM 内存结构：堆、栈、方法区、元空间、直接内存
2. 垃圾回收：判断对象存活、GC 算法、GC 流程
3. 垃圾收集器：CMS、G1、ZGC 特点与适用场景
4. 类加载机制、双亲委派、打破双亲委派
5. OOM、内存泄漏、GC 日志、JVM 参数调优

## 五、Spring 全家桶 + **源码&原理考察**
1. Spring IOC 容器启动流程
2. Bean 完整生命周期
3. 循环依赖解决方案（三级缓存）
4. AOP 原理：JDK 动态代理 / CGLIB
5. Spring Transaction 事务原理、传播行为、隔离级别
6. SpringBoot 自动配置原理、starter 机制

## 六、数据库 MySQL（重中之重）

1. 索引结构 B+Tree、聚簇/非聚簇索引
2. 最左前缀、回表、索引覆盖、索引失效
3. 事务 ACID、隔离级别、MVCC 原理
4. 锁：行锁、表锁、间隙锁、临键锁
5. SQL 优化、explain、慢查询优化
6. 大表优化、分库分表基础

## 七、缓存 & 消息队列 & 分布式中间件
1. Redis：数据结构、持久化、集群、过期策略
2. 分布式锁、Redlock、Redisson
3. MQ：RabbitMQ/Kafka 消息可靠性、重复消费、丢失、顺序
4. 分布式基础：CAP、BASE、最终一致性
5. 分布式事务、Seata 等

## 八、**业务场景题（独立大模块，决定通过率）**
**纯实战，不考背诵，面试官最爱**
1. 高并发：秒杀、超卖、限流、熔断、降级
2. 缓存：雪崩、穿透、击穿、热点Key、双写一致性
3. 接口：幂等性设计、防重、防刷
4. 订单：支付、取消、超时关单
5. 库存：扣减超卖、最终一致性
6. 大数据量：分页、大表、批量导入导出
7. 高可用：重试、兜底、降级方案

## 九、系统设计 / 架构设计（高级必问）
1. 单体 → 微服务拆分思路
2. 接口设计、表结构设计
3. 高并发系统设计
4. 高可用设计
5. 常见业务系统设计（订单、支付、优惠券、登录）

## 十、设计模式
1. 单例（双重校验、静态内部类、枚举）
2. 工厂、抽象工厂
3. 代理、装饰器、适配器
4. 策略、模板方法、观察者
5. 实际项目中如何使用

## 十一、算法与数据结构
1. 数组、链表、栈、队列、二叉树
2. 哈希、堆、排序
3. 双指针、滑动窗口、DFS/BFS
4. 动态规划（简单~中等）
5. 常见 LeetCode 高频题

## 十二、项目经验 + 软素质
1. 项目业务描述、职责、亮点、难点
2. 技术选型理由、性能优化
3. 线上问题排查、事故处理
4. 学习能力、表达逻辑、沟通态度
