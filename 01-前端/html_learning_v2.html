HTML 全局属性
全局属性可用于任何 HTML 元素，是控制行为、样式和无障碍访问的基础工具。
核心标识属性 定义元素身份
属性名类型说明示例
idstring唯一标识符，页面内不可重复，用于 CSS 选择器和 JS 定位id="header"
classstringCSS 类名，可有多个（空格分隔），用于样式和 JS 选择class="btn primary"
namestring表单元素名称，提交时作为数据键名name="email"
titlestring附加说明信息，鼠标悬停时显示为工具提示title="点击提交"
样式属性 外观控制
属性名类型说明示例
styleCSS内联 CSS 样式，优先级最高，建议少用（维护性差）style="color:red"
hiddenboolean隐藏元素，等效于 display:none，但保留 DOM 结构hidden
数据属性 自定义数据
属性名类型说明示例
data-*any自定义数据属性，* 可任意命名，通过 dataset 在 JS 中读取data-user-id="42"
JS 读取示例：
const el = document.querySelector('[data-user-id]');
console.log(el.dataset.userId); // "42"
无障碍属性（ARIA） 可访问性
属性名类型说明示例
aria-labelstring为无文本内容的元素提供可读标签（供屏幕阅读器读出）aria-label="关闭"
aria-hiddenbooleantrue 表示对辅助技术隐藏（装饰性图标等）aria-hidden="true"
aria-expandedboolean表示折叠/展开元素的当前状态（如下拉菜单）aria-expanded="false"
rolestring定义元素的语义角色，补充或覆盖默认语义role="button"
tabindexnumber控制 Tab 键焦点顺序，0=可聚焦，-1=不可 Tab 聚焦tabindex="0"
交互属性 行为控制
属性名类型说明示例
contenteditablebooleantrue 使该元素内容可被用户直接编辑contenteditable="true"
draggablebooleantrue 使该元素可被拖拽（配合 drag 事件 API 使用）draggable="true"
spellcheckboolean控制是否进行拼写检查spellcheck="false"
translateyes/nono 阻止翻译工具翻译该内容（如品牌名、代码）translate="no"
langstring声明元素内容的语言，影响辅助技术和搜索引擎lang="zh-CN"
dirltr/rtl文字方向，ltr 从左到右，rtl 从右到左（阿拉伯语等）dir="rtl"
