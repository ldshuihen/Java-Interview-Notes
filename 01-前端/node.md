# Node.js 面试核心学习清单（2026最新版）
我给你整理了**最实用、面试必问、企业真实考察**的 Node.js 知识点，按**优先级从高到低**排序，直接照着学就能应对 80% 以上的面试。

## 一、基础核心（必学，面试 50% 考点）
### 1. Node.js 是什么 & 运行原理
- 基于 Chrome V8 引擎的 JavaScript 运行时
- **非阻塞 I/O**、**事件驱动**
- 单线程 + 多线程（libuv 线程池）

### 2. 事件循环（Event Loop）【高频面试题】
- 6 个阶段执行顺序
- `microtask`（微任务）和 `macrotask`（宏任务）执行优先级
- `process.nextTick` / `Promise` / `setTimeout` 执行顺序

### 3. 模块化
- CommonJS 规范（`require` / `module.exports`）
- ES Module（`import` / `export`）
- 两者区别、循环引用问题

### 4. 全局对象 & 常用模块
- `global`、`process`、`__dirname`、`__filename`
- 内置模块：**fs、path、http、url、querystring、crypto**

---

## 二、异步编程（必考）
### 1. 异步方案演进
- 回调函数 → Promise → async/await
- 回调地狱如何解决

### 2. Promise 深入
- 三种状态
- 手写 Promise（部分大厂面试）
- `Promise.all` / `race` / `allSettled` 用法

### 3. async/await 语法与错误处理
- `try/catch`
- 同步写法处理异步逻辑

---

## 三、Web 开发（Node 面试核心）
### 1. http 模块
- 手写 HTTP 服务器
- req/res 对象、请求头、响应头、状态码

### 2. Express / Koa / Nest.js（至少精通一个）
**Express**
- 中间件机制
- 路由、参数获取、错误处理
- 静态资源服务

**Koa**
- 洋葱模型（必问）
- async/await 中间件

**Nest.js**（现在大厂非常爱考）
- 模块化、依赖注入 DI
- 控制器、服务、过滤器、拦截器

### 3. 中间件原理
- 什么是中间件
- 手写简单中间件

---

## 四、数据库与缓存
### 1. MongoDB / MySQL

### 2. Redis 缓存

- 缓存雪崩、击穿、穿透（面试高频）

## 五、工程化 & 部署
### 1. 包管理

- package.json 配置
- 依赖版本号规则（^ ~）

### 2. 项目部署

### 3. 跨域、CORS、代理

- Node 端解决跨域方案
