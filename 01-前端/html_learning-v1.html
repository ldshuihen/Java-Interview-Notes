HTML 标签分类总览
HTML5 共有约 100+ 个标签，按功能分为以下 7 大类别。点击分类可快速跳转学习。
<!-- HTML 文档基本骨架 -->
<!DOCTYPE html>
<html lang="zh-CN">
  <head>← 元信息区域（不显示）
    <meta charset="UTF-8">
    <title>页面标题</title>
  </head>
  <body>← 页面内容区域（可见）
    <header>页头</header>
    <main>主内容</main>
    <footer>页脚</footer>
  </body>
</html>

文档结构标签
构建页面骨架的基础标签，定义页面的整体层次与布局容器。
<html>
根元素
HTML 文档的根标签，所有内容都包裹在其中。lang 属性声明语言。
<html lang="zh-CN">
  <head>...</head>
  <body>...</body>
</html>
▶ 点击展开示例
<head>
文档头部
包含页面元信息，不在页面中显示。内含 title、meta、link 等标签。
<head>
  <meta charset="UTF-8">
  <title>我的网站</title>
  <link rel="stylesheet" href="style.css">
</head>
▶ 点击展开示例
<body>
页面主体
包含所有页面可见内容。浏览器渲染的核心区域，支持大量事件属性。
<body onload="init()">
  <!-- 所有可见内容放这里 -->
  <h1>Hello World</h1>
</body>
▶ 点击展开示例
<div>
块级容器
通用块级容器，无语义含义，常与 class/id 配合用于布局和样式控制。
<div class="card">
  <div class="card-body">
    内容区域
  </div>
</div>
▶ 点击展开示例
<span>
行内容器
通用行内容器，无语义含义，常用于对文本片段应用样式或脚本。
<p>价格：<span class="price">¥99</span></p>

<!-- span 不换行，div 换行 -->
▶ 点击展开示例
<hr> / <br>
分隔与换行
hr 渲染水平分割线；br 插入强制换行。两者均为空元素（无闭合标签）。
第一段文字<br>换行后继续

<hr><!-- 分割线 -->

第二段落
▶ 点击展开示例
📝
文本内容标签
处理文字排版、强调、链接、列表等内容的标签集合。
<h1> ~ <h6>
标题层级
六级标题，h1 最大最重要，h6 最小。对 SEO 和无障碍访问非常关键，每页只应有一个 h1。
<h1>一级标题（页面主标题）</h1>
<h2>二级标题（章节）</h2>
<h3>三级标题（子章节）</h3>
<!-- 不要跳级使用，保持层级顺序 -->
▶ 点击展开示例
<p>
段落
定义段落文本，浏览器在段落前后自动添加空白间距。是最常用的文本容器。
<p>这是第一段，浏览器会自动
处理段落间距。</p>
<p>这是第二段。</p>
▶ 点击展开示例
<a>
超链接
创建超链接，href 指定目标地址，target="_blank" 在新标签打开，rel="noopener" 提高安全性。
<!-- 外部链接 -->
<a href="https://example.com"
   target="_blank"
   rel="noopener">
  访问示例网站
</a>

<!-- 页内锚点 -->
<a href="#section1">跳转到第一节</a>
▶ 点击展开示例
<strong> / <em>
强调文本
strong 表示重要性（粗体），em 表示强调（斜体）。两者都有语义含义，不同于纯样式的 b/i。
<p>
  <strong>警告：</strong>
  请<em>认真</em>阅读说明。
</p>

<!-- strong = 重要; em = 强调 -->
▶ 点击展开示例
<ul> / <ol> / <li>
列表
ul 无序列表（项目符号），ol 有序列表（编号），li 为列表项。可嵌套使用。
<ul><!-- 无序 -->
  <li>苹果</li>
  <li>香蕉</li>
</ul>

<ol start="1"><!-- 有序 -->
  <li>第一步</li>
  <li>第二步</li>
</ol>
▶ 点击展开示例
<code> / <pre>
代码展示
code 表示行内代码片段；pre 保留空白和换行，常与 code 配合展示代码块。
<!-- 行内代码 -->
使用 <code>console.log()</code> 输出

<!-- 代码块 -->
<pre><code>
function hello() {
return "Hi!";
}
</code></pre>
▶ 点击展开示例
🖼️
多媒体标签
嵌入图片、音频、视频及可编程画布的标签集合。
<img>
图片
嵌入图片，src 为图片路径，alt 为替代文本（无障碍访问必需），可设置 width/height 防止布局抖动。
<img
  src="photo.jpg"
  alt="一只可爱的猫咪"
  width="400"
  height="300"
  loading="lazy"
>
<!-- loading=lazy 延迟加载 -->
▶ 点击展开示例
<video>
视频
嵌入视频，支持 controls 显示播放控件，autoplay 自动播放，muted 静音（自动播放必需）。
<video
  src="movie.mp4"
  controls
  width="640"
  poster="cover.jpg"
>
  浏览器不支持视频播放
</video>
▶ 点击展开示例
<audio>
音频
嵌入音频文件，controls 显示播放器，loop 循环播放，可使用多个 source 标签提供不同格式。
<audio controls loop>
  <source src="song.mp3"
         type="audio/mpeg">
  <source src="song.ogg"
         type="audio/ogg">
</audio>
▶ 点击展开示例
<canvas>
画布
通过 JavaScript API 绘制 2D/3D 图形的画布元素，用于图表、动画、游戏开发。
<canvas id="myCanvas"
         width="500"
         height="300">
</canvas>

// JS 获取画布上下文：
const ctx = canvas.getContext('2d');
ctx.fillRect(0, 0, 100, 100);
▶ 点击展开示例
<picture>
响应式图片
为不同屏幕尺寸或设备提供多种图片资源，浏览器自动选择最合适的。
<picture>
  <source
    media="(min-width:768px)"
    srcset="large.jpg">
  <source
    media="(min-width:480px)"
    srcset="medium.jpg">
  <img src="small.jpg" alt="图片">
</picture>
▶ 点击展开示例
<iframe>
内联框架
在页面中嵌入另一个网页，常用于嵌入地图、视频播放器等第三方内容。
<iframe
  src="https://example.com"
  width="600"
  height="400"
  title="嵌入内容描述"
  loading="lazy"
></iframe>
▶ 点击展开示例
📋
表单交互标签
收集用户输入、实现人机交互的表单元素集合。
<form>
表单容器
定义表单区域，action 指定提交地址，method 指定提交方式（get/post）。
<form
  action="/submit"
  method="post"
  enctype="multipart/form-data"
>
  <!-- 表单元素放这里 -->
  <button type="submit">提交</button>
</form>
▶ 点击展开示例
<input>
输入框
最通用的表单控件，type 属性决定类型：text/email/password/number/checkbox/radio/file/date 等。
<!-- 文本 -->
<input type="text" placeholder="姓名">

<!-- 邮箱（自动验证格式）-->
<input type="email" required>

<!-- 复选框 -->
<input type="checkbox" checked>记住我

<!-- 文件上传 -->
<input type="file" accept="image/*">
▶ 点击展开示例
<button>
按钮
可点击按钮，type 为 submit/reset/button。支持内嵌 HTML 内容，比 input[type=button] 更灵活。
<!-- 提交按钮 -->
<button type="submit">登录</button>

<!-- 普通按钮（JS控制）-->
<button type="button"
       onclick="save()">
  💾 保存
</button>

<!-- 禁用状态 -->
<button disabled>不可用</button>
▶ 点击展开示例
<select>
下拉选择
创建下拉菜单，option 为选项，optgroup 分组选项，multiple 支持多选。
<select name="city">
  <optgroup label="华南">
    <option value="gz">广州</option>
    <option value="sz" selected>深圳</option>
  </optgroup>
</select>
▶ 点击展开示例
<textarea>
多行文本
多行文本输入框，rows/cols 设置初始大小，可通过 CSS resize 控制是否允许调整大小。
<textarea
  name="comment"
  rows="5"
  cols="40"
  placeholder="请输入评论..."
  maxlength="500"
></textarea>
▶ 点击展开示例
<label>
表单标签
为表单控件提供说明文字，for 属性绑定对应 input 的 id，点击 label 可聚焦/触发对应控件。
<!-- 方式1：for + id 绑定 -->
<label for="email">邮箱地址</label>
<input id="email" type="email">

<!-- 方式2：包裹绑定 -->
<label>
  记住密码
  <input type="checkbox">
</label>
▶ 点击展开示例
📊
表格数据标签
用于展示结构化数据的表格元素，注意表格应仅用于数据展示，不应用于页面布局。
<table>
表格完整结构
table 为容器，thead/tbody/tfoot 语义分组，tr 为行，th 为表头单元格，td 为数据单元格。
<table>
  <thead>
    <tr>
      <th>姓名</th>
      <th>年龄</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>张三</td>
      <td>25</td>
    </tr>
  </tbody>
</table>
▶ 点击展开示例
colspan / rowspan
单元格合并
colspan 合并列，rowspan 合并行，实现复杂表格布局。合并后被覆盖的单元格需删除。
<table>
  <tr>
    <!-- 横跨 2 列 -->
    <td colspan="2">合并两列</td>
  </tr>
  <tr>
    <!-- 纵跨 2 行 -->
    <td rowspan="2">合并两行</td>
    <td>B</td>
  </tr>
</table>
▶ 点击展开示例
<caption>
表格标题
为表格提供标题或说明，必须作为 table 的第一个子元素，对无障碍访问很重要。
<table>
  <caption>2024年销售数据统计</caption>
  <thead>...</thead>
  <tbody>...</tbody>
</table>
▶ 点击展开示例
🔖
语义化标签（HTML5）
HTML5 新增的语义化结构标签，让代码更具可读性，对 SEO 和无障碍访问至关重要。
<header>
页头 / 区块头
表示页面或区块的头部，通常包含导航、logo、搜索栏。可在 article/section 内重复使用。
<header>
  <nav>
    <a href="/">首页</a>
    <a href="/about">关于</a>
  </nav>
</header>
▶ 点击展开示例
<nav>
导航区域
定义导航链接集合，用于主导航菜单、面包屑导航等。屏幕阅读器会识别此区域。
<nav aria-label="主导航">
  <ul>
    <li><a href="/">首页</a></li>
    <li><a href="/blog">博客</a></li>
  </ul>
</nav>
▶ 点击展开示例
<main>
主内容区
页面的主要内容区域，每个页面只能有一个 main 元素，不应包含重复性内容如导航和页脚。
<body>
  <header>...</header>
  <main>
    <!-- 页面核心内容 -->
    <h1>文章标题</h1>
    <article>...</article>
  </main>
  <footer>...</footer>
</body>
▶ 点击展开示例
<article>
独立内容块
表示独立的内容块，可脱离页面上下文独立分发，如博客文章、新闻、评论、论坛帖子。
<article>
  <header>
    <h2>博客标题</h2>
    <time datetime="2024-01-15">
      2024年1月15日</time>
  </header>
  <p>文章内容...</p>
</article>
▶ 点击展开示例
<section>
内容章节
定义文档中的通用章节，通常有标题。与 div 的区别在于 section 代表有主题的内容分组。
<section id="features">
  <h2>产品特点</h2>
  <p>本产品具有以下优势...</p>
</section>

<section id="pricing">
  <h2>价格方案</h2>
</section>
▶ 点击展开示例
<aside> / <footer>
侧栏与页脚
aside 表示与主内容相关但独立的内容（侧边栏、广告、引用）；footer 表示页脚，含版权、联系信息等。
<aside>
  <h3>相关文章</h3>
  <ul>...</ul>
</aside>

<footer>
  <p>© 2024 公司名称</p>
  <nav>底部导航</nav>
</footer>
▶ 点击展开示例
⚙️
元信息标签
位于 <head> 内的标签，定义文档元数据，控制字符集、SEO、样式和脚本加载。
<meta>
元数据
定义文档元数据，包括字符集、视口、SEO 描述、Open Graph 社交分享等关键配置。
<!-- 字符集 -->
<meta charset="UTF-8">

<!-- 响应式视口 -->
<meta name="viewport"
      content="width=device-width,
              initial-scale=1">

<!-- SEO 描述 -->
<meta name="description"
      content="页面描述...">
▶ 点击展开示例
<link>
外部资源链接
链接外部资源，最常用于引入 CSS 样式表，也用于网站图标、预加载资源等。
<!-- CSS 样式表 -->
<link rel="stylesheet"
      href="style.css">

<!-- 网站图标 -->
<link rel="icon"
      href="favicon.ico">

<!-- 预加载关键资源 -->
<link rel="preload"
      href="hero.jpg"
      as="image">
▶ 点击展开示例
<script>
JavaScript 脚本
嵌入或引用 JS 脚本。defer 延迟执行（推荐），async 异步加载，type="module" 支持 ES 模块。
<!-- 外部脚本（推荐放 body 末）-->
<script src="app.js" defer></script>

<!-- ES 模块 -->
<script type="module">
  import { init } from './main.js';
  init();
</script>
▶ 点击展开示例
<style>
内嵌样式
在 HTML 文档中直接嵌入 CSS 样式。适合小型页面，大型项目建议使用外部 link 引入。
<style>
  body {
    font-family: sans-serif;
    background: #f0f0f0;
  }
  .highlight {
    color: red;
  }
</style>
